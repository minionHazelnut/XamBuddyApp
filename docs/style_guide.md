# XamBuddyRN Style Guide

## Color palette
- Primary background: `#EBFFF4`
- Top header / surface: `#5e7070`
- Card surface: `#ffffff`
- Accent card: `#dff0ea`
- Text primary: `#000000`
- Text secondary / subtitle: `rgba(0,0,0,0.67)`
- Accent text / icon: `#4a5568`
- Dark text / strong accent: `#2d3748`
- Mint decorative tone: `rgba(235,255,244,0.35)`
- Gradient stops: `['#f2fdf7', '#EBFFF4']`

## Typography
- Headings: `IstokWeb-Regular`
- Bold headings: `IstokWeb-Bold`
- Body / paragraph: `IowanOldStyle-Roman`
- Bold body / emphasis: `IowanOldStyle-Bold`

## UI patterns
- Use soft green/mint theme across backgrounds and cards.
- Use white rounded cards with subtle shadows and elevation.
- Keep section spacing generous and consistent.
- Use rounded corners on cards and buttons.
- Use gradient backgrounds for large app sections where applicable.
- Keep icons minimal and color-contrasted with dark accents.

## Consistency rules for future screens
- Match `HomeScreen` card styling and spacing for new cards.
- Reuse the same text color tokens for titles and subtitles.
- Preserve the same border radius and elevation style across cards.
- Use the same font tokens from `lib/fonts.js` in all UI screens.

## Notes
- This guide is based on the current `HomeScreen` visual language.
- Future UI screens should follow these colors, typography, and card patterns.
